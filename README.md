# XAUUSD 历史行情数据集

> 🌐 语言切换：**中文** | [English](./README_EN.md)

---

## 简介

本仓库收录了 **17 个金融品种** 的 H1（1小时）历史 OHLCV 行情数据，数据来源于 MetaTrader 5，最早可追溯至 2007 年，截止至 2026 年 4 月，合计超过 **113 万行**。

---

## 文件结构

文件命名规则：`{品种代码}_H1.csv`

### 数据列说明

| 列名 | 类型 | 说明 |
|---|---|---|
| `time` | datetime | K线开盘时间（UTC） |
| `open` | float | 开盘价 |
| `high` | float | 最高价 |
| `low` | float | 最低价 |
| `close` | float | 收盘价 |
| `tick_volume` | int | Tick 成交量（成交量代理指标） |

### 示例数据

```
time,open,high,low,close,tick_volume
2007-06-22 03:00:00,651.3,656.3,650.6,653.9,881
```

## 快速开始（Python）

```python
import pandas as pd

# 读取黄金数据
df = pd.read_csv("XAUUSD_H1.csv", parse_dates=["time"], index_col="time")

print(df.head())
print(f"总行数：{len(df)}")
print(f"时间范围：{df.index.min()} → {df.index.max()}")
```

---

## 注意事项

- 数据由 **MetaTrader 5** Python 接口导出
- 时间周期为 **H1**，每行代表 1 小时
- `tick_volume` 为 Tick 计数，非真实成交量，是外汇市场常用的成交量代理指标
- 部分品种行数较少，受限于券商历史数据深度
- 所有时间戳均为 **UTC**
